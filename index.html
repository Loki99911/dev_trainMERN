=== GIT ===

            Что б добавить рабочий код на репозиторий клиента
        надо сделать несколько операций в консоли:
        1)cd /путь/к/папке/для/проекта
        2)git clone git@example.com:example/test.git
        3)Имя деректории предположим - "test".
          cd test
        4)git init
        5)git remote add client git@example.com:example/test.git
        6)git add .
        7)git commit -m "Ваше сообщение о коммите здесь"
        8)git push client master
        Или проделать подобные операции в GitHub Desktop:)
